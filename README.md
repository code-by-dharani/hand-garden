# 🌿 Hand Garden

Hand Garden is a simple and elegant garden-themed web page built using HTML. The project focuses on creating a clean, visually appealing interface with a nature-inspired design.

---

## ✨ Features

- 🌱 Clean and attractive UI
- 📱 Responsive design
- 🌿 Nature-inspired layout
- ⚡ Lightweight and fast
- 💻 Easy to customize

---

## 🛠️ Technologies Used

- HTML5

---

## 📂 Project Structure

```text
hand-garden/
└── hand-garden.html
```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/code-by-dharani/hand-garden.git
```

### Run the Project

Open `hand-garden.html` in your preferred web browser.
